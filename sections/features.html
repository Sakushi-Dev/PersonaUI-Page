Features

Everything You Need to Create Living AI

PersonaUI brings together personality, memory, and emotion in a single local-first desktop app.

Cortex Memory System

Three-file memory architecture — Memory, Soul, Relationship — updated autonomously via tool use at configurable context thresholds.

Afterthought System

The AI independently decides whether to send a follow-up message. A 10-second timer lets you cancel if needed.

8-Step Onboarding

Guided first-launch setup covering your profile, API key, context settings, Cortex config, and UI preferences.

Slash Commands

Type / in the chat to access commands with autocomplete and keyboard navigation — extensible for frontend and backend.

36 Prompt Templates

JSON-based personality templates with three-phase placeholder resolution. A standalone Prompt Editor lets you inspect and edit them.

Network Sharing

Share access over your local network with IP-based access control and QR code for quick mobile device connection.

19 Overlay Dialogs

In-app panels for API settings, Cortex editing, avatar management, persona configuration, and much more.

Internationalization

Full i18n support with feature-scoped locale files — currently available in English and German.

Complete Privacy

All data lives exclusively on your machine. Nothing is sent to external servers beyond the AI model requests themselves.
