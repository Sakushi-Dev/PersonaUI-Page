Quick Start

Get Running in Seconds

Clone, run, done. The launcher handles everything for you.

$ git clone https://github.com/Sakushi-Dev/PersonaUI.git && cd PersonaUI && .\bin\start.bat

# Clone the repository

$ git clone https://github.com/Sakushi-Dev/PersonaUI.git

# Navigate to the folder

$ cd PersonaUI

# Launch — auto-installs Python 3.12, Node.js, builds frontend

$ bin\start.bat
$ git clone https://github.com/Sakushi-Dev/PersonaUI.git
$ cd PersonaUI

# Create & activate virtual environment

$ python -m venv .venv
$ .venv\Scripts\activate

# Install & run

$ pip install -r requirements.txt
$ python src/init.py

Works on Windows. Installs Python 3.12, Node.js, and all dependencies for you.
